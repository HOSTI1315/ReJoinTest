# Rejoin-Mumu — Roblox Rejoin для MuMu эмуляторов

Специализированный реджойн-инструмент для Android-эмуляторов MuMu.

## Функции
- Сканирование инстансов MuMu
- Маппинг ADB-портов на Roblox PlaceID
- Мониторинг состояния Roblox в каждом эмуляторе
- Авто-перезапуск при зависании

## Структура
- Точка входа: `main.py`
- Конфиг: `config.json` (пути к MuMu)
- Маппинг портов: `port_map.json`

## Запуск
```
python main.py
```

## Стек
Python (стандартная библиотека)

## Статус
Специализированная версия для работы через эмуляторы.
