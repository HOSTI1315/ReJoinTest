# Rejo — Базовый Roblox Rejoin Tool

Консольный инструмент для автоматического поддержания Roblox-аккаунтов в игре.

## Функции
- Несколько аккаунтов одновременно
- Авто-перезапуск при вылете/краше
- Поддержка приватных серверов
- Использует `.ROBLOSECURITY` куки

## Структура
- Точка входа: `Rejointool.py`
- Конфиг: `rejoin_config.json` (создаётся при первом запуске, хранит куки аккаунтов)

## Запуск
```
python Rejointool.py
```

## Стек
Python (requests, psutil, win32event)

## Статус
Первая, базовая версия реджойн-инструмента.
